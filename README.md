# 🧠 AI Second Brain

A FastAPI-powered backend that helps store, organize, and retrieve knowledge efficiently.

## 🚀 Features

* Store notes & files
* Semantic search (coming soon)
* AI-powered retrieval

## 🛠 Tech Stack

* FastAPI
* SQLAlchemy
* ChromaDB (planned)
* React (frontend)

## ⚙️ Setup

```bash
git clone https://github.com/your-username/Second_brain.git
cd Second_brain/backend

pip install -r requirements.txt
uvicorn app.main:app --reload
```
